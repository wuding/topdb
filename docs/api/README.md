# Topdb



## 依赖

### 基本

| 方法名       | 描述                   |
| ------------ | ---------------------- |
| __construct  | 配置和属性，初始化     |
| __call       | 调用 PDO 扩展方法      |
| __callStatic | 转接到动态方法         |
| init         | 连接（参数），内存对象 |



### 连接

| 方法名  | 描述                   |      |
| ------- | ---------------------- | ---- |
| connect | 根据参数对应的连接实例 |      |
| conf    | PDO 新建对象所需参数   |      |
| dns     |                        |      |



### 内存缓存

| 方法名 | 描述 |
| ------ | ---- |
| mem    | 对象 |
